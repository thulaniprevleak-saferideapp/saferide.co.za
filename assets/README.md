This directory contains all media assets for the SafeRide web application.

### Hero Section
- **hero-phone.png**: Phone mockup showing SafeRide app interface

### App Store Badges
- **google-play.png**: Google Play Store badge
- **app-store.png**: Apple App Store badge

## Asset Guidelines

### Logo
- Format: SVG (scalable) or PNG with transparent background
- Size: Minimum 200x200px for PNG
- Colors: Primary blue (#1e40af) with accent colors

### Hero Image
- Format: PNG with transparency
- Recommended size: 800x1200px
- Should show app interface clearly

### App Store Badges
- Use official badges from:
  - Google Play: https://play.google.com/intl/en_us/badges/
  - Apple App Store: https://developer.apple.com/app-store/marketing/guidelines/
